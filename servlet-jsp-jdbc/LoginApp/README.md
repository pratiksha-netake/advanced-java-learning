\# LoginApp



\## project description

This is a simple login and registration system built using :

* java servlet
* jsp
* jdbc
* MySQL



\## Features 

* user Registartion
* user Login
* session Management
* Logout functionality
* form validation



\## Technologies Used

* java
* servlet API
* JSP
* JDBC
* MySQL
* Apache Tomcat
