# Library Management System

## project Overview
This is console-based Library Management System devloped using **java and  JDBC **.
It allowas management of booksc,authors,categories ,users and transactions.

## Technologies used
- Java
- JDBC
- MySQL
- SQL

## features 
- Add Author
- Add Book
- View Book
- Add User
- Issue Book
- Return Book

## Database Design

Tables Used:
- Authors
- Categories
- Books
- Users
- Transactions

Concepts:
- Foreign Keys
- Relationships
- Constraints
- SQL Queries
